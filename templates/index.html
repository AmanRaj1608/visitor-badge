How to use?

If you know how to add picture in markdown, then you are good to go.

All you have to do is to add below markdown markup to your markdown content: 

                
                    ![visitors](https://visistor-badge.glitfh.me/badge)

You can use it in your

  • README.md(or any other .md files)
  • Issue body
  • Pull request description

and any other markdown content or even your html files you want.

FAQ

Can I use this in HTML file?

Sure!

All you have to do is to insert a <img> to your HTML:

                    
                        <img src="https://visitor-badge.glitch.me" alt="visitor badge"/>
